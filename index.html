Sobre o Projeto

O Contest Guardian é a plataforma criada para quem enxerga o estudo como um campo de batalha. Cada simulado, relatório e mensagem motivacional são parte do treinamento. Com inteligência artificial, o sistema adapta o plano de estudos ao seu ritmo e ao seu objetivo.

Tecnologias Utilizadas

Spring / .NET
React + Tailwind
TensorFlow / PyTorch
PostgreSQL
